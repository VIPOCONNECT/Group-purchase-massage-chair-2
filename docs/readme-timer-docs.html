📚 תיעוד מערכת טיימר קמפיין - למפתחים

🎯 סקירה כללית

מערכת טיימר קמפיין היא מערכת פשוטה ויעילה להצגת טיימר יורד לסיום קמפיין, עם ממשל ניהול נוח למנהלים.

המערכת בנויה בטכנולוגיות בסיסיות (HTML, CSS, JavaScript) ללא תלות בספריות חיצוניות.

📁 מבנה הקבצים

  • index.html - דף הצגת הטיימר ללקוחות
  • admin.html - דף ניהול הקמפיין למנהלים
  • timer.js - לוגיקת הטיימר והצגה
  • admin.js - ניהול תאריכים וכותרות
  • style.css - עיצוב אחיד לכל הדפים

⚙️ הגדרות אחסון

המערכת משתמשת ב-localStorage לשמירת הנתונים:

campaignEndDate - תאריך סיום הקמפיין בפורמט ISO String campaignTitle - כותרת הקמפיין (ברירת מחדל: "טיימר קמפיין")

🔧 API פנימי - timer.js

ה-API מחשף דרך אובייקט window.TimerAPI:

מתודה תיאור החזרה
getCampaignEndDate() מחזיר את תאריך הסיום מה-localStorage Date object או null
getCampaignTitle() מחזיר את כותרת הקמפיין מחרוזת
calculateTimeRemaining(endDate) מחשב את הזמן שנותר עד תאריך הסיום אובייקט עם ימים, שעות, דקות, שניות
updateDisplay(time) מעדכן את התצוגה עם הזמן החדש void
updateTimer() מעדכן את כל הטיימר (קריאה ידנית) void

🎨 התאמה אישית

ניתן להתאים את המערכת בקלות:

הוספת פונקציות חדשות

טיפ: ניתן להרחיב את ה-API בקלות על ידי הוספת מתודות חדשות לאובייקט window.TimerAPI.
